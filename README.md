# Android Design System

A reusable Android Design System built with Jetpack Compose.

This project demonstrates how to create scalable, reusable UI components that can be shared across multiple features and applications while maintaining visual consistency, accessibility, and developer productivity.

## Features

- Jetpack Compose components
- Material 3 theming
- Light & Dark mode support
- Accessibility-first design(TODO)
- Component Catalog / Showcase
- Reusable design tokens
- Preview infrastructure

## Components

Current components include:

- Buttons
  - Primary Button
  - Secondary Button
  - Disabled States

- Top Bars(TODO)
  - Center Aligned Top Bar
  - Navigation Top Bar

- Bottom Sheets(TODO)
  - Modal Bottom Sheet
  - Action Bottom Sheet

More components will be added over time.
